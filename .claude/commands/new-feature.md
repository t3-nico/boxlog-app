# 新機能実装

新機能を実装する前に、以下のワークフローに従ってください。

## Explore → Plan → Code → Commit

### 1. Explore（探索）
- 関連する既存コードを検索
- 類似機能がないか確認
- 影響範囲を把握

### 2. Plan（計画）
以下を明確にしてから実装開始：
- [ ] 実装するファイル一覧
- [ ] 使用するコンポーネント・hooks
- [ ] 必要な型定義
- [ ] テストケース

### 3. Code（実装）
CLAUDE.md準拠で実装：
- セマンティックトークン使用
- 8pxグリッド準拠
- i18n対応（ハードコード禁止）
- 型安全（any禁止）

### 4. Commit（コミット）
- `npm run typecheck` 実行
- `npm run lint` 実行
- 適切なコミットメッセージ

## 出力形式

```markdown
## 実装計画

### 影響範囲
- [ファイル一覧]

### 実装ステップ
1. [ステップ1]
2. [ステップ2]

### 確認事項（開発者へ）
- [確認が必要な点]
```
