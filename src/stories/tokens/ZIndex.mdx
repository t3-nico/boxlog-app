# Z-Index レイヤー

UI コンポーネントのスタッキング順序を一元管理。値が大きいほど前面に表示される。

---

## 使用方法

```tsx
// ✅ Tailwind クラスとして使用
<div className="z-modal">...</div>

// ❌ CSS 変数として使用（非推奨）
style={{ zIndex: 'var(--z-index-modal)' }}

// ❌ 任意値（禁止）
<div className="z-[200]">...</div>
```

---

## レイヤー一覧

| 値 | Tailwind クラス | 名前 | 用途 |
|----|----------------|------|------|
| 50 | `z-dropdown` | dropdown | ドロップダウンメニュー、セレクト |
| 100 | `z-popover` | popover | ポップオーバー（日付選択、カラーピッカー） |
| 150 | `z-sheet` | sheet | サイドシート、ドロワー（Inspector 等） |
| 200 | `z-modal` | modal | 通常のダイアログ・モーダル |
| 250 | `z-confirm` | confirm | 確認ダイアログ（削除など重要な操作） |
| 300 | `z-toast` | toast | トースト通知 |
| 350 | `z-context-menu` | context-menu | コンテキストメニュー（右クリック） |
| 1000 | `z-calendar-drag` | calendar-drag | カレンダードラッグプレビュー |
| 1099 | `z-inspector-backdrop` | inspector-backdrop | ドラッグ可能 Inspector 背景 |
| 1100 | `z-inspector` | inspector | ドラッグ可能 Inspector |
| 1200 | `z-overlay-popover` | overlay-popover | Inspector 上の日付選択・カラーピッカー |
| 1300 | `z-overlay-modal` | overlay-modal | Inspector 上のモーダル |
| 1400 | `z-overlay-confirm` | overlay-confirm | Inspector 上の確認ダイアログ |
| 9999 | `z-tooltip` | tooltip | ツールチップ（最前面） |

---

## レイヤーグループ

### 通常コンテキスト（50-350）

ページ上の通常のオーバーレイ要素。dropdown, popover, sheet, modal, confirm, toast, context-menu

### Inspector（1000-1100）

ドラッグ操作やフローティング UI。calendar-drag, inspector-backdrop, inspector

### Overlay 系（1200-1400）

Inspector 上の UI。overlay-popover, overlay-modal, overlay-confirm

### 最前面（9999）

常に最前面に表示される要素。tooltip

---

## よくあるパターン

| シナリオ | 使用クラス |
|---------|-----------|
| Inspector 上のモーダル | `z-overlay-modal` |
| Inspector 上のモーダル内の日付選択 | 自動的に上に表示される（ポータル経由） |
| Inspector 上の確認ダイアログ | `z-overlay-confirm` |
| 通常の確認ダイアログ | `z-confirm` |

---

## 基本原則

- **Tailwind クラスを使用**: `z-modal`, `z-tooltip` など
- **任意値は避ける**: `z-[200]` ではなく `z-modal`
- **Inspector 上の UI は overlay 系**: Inspector 上のモーダルは `z-overlay-modal`、日付選択は `z-overlay-popover`
