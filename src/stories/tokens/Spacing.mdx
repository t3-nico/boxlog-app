# Spacing（余白）

要素間の関係性に基づいて Spacing を決定。近いほど関連が強い。

---

## 原則

- **近い** → 強く関連（同じグループ）
- **遠い** → 弱い関連（異なるグループ）
- **内→外** → 内側は狭く、外側は広く（入れ子の四角形）

---

## Spacing スケール（8pt グリッド）

8px を基本単位とし、8の倍数のみを使用。

| Tailwind値 | ピクセル | 名前 | 用途 |
|------------|---------|------|------|
| `1` | 4px | XS | アイコンとテキスト間 |
| **`2`** | **8px** | **S** | **標準（同グループ内）** |
| **`4`** | **16px** | **M** | **標準（グループ間）** |
| **`6`** | **24px** | **L** | **セクション間** |
| **`8`** | **32px** | **XL** | **ページ余白** |
| `12` | 48px | 2XL | 大きなセクション区切り |
| `16` | 64px | 3XL | ページ区切り |

---

## Gap（要素間の余白）

Flexbox / Grid での要素間余白。関係性に応じて選択。

| クラス | 用途 |
|--------|------|
| `gap-1` | タイト（アイコン+テキスト） |
| `gap-2` | 標準（同グループ内、ボタン群など） |
| `gap-4` | グループ間、フォームフィールド |
| `gap-6` | セクション間 |
| `gap-8` | 大きなセクション区切り |

---

## Padding（内側余白）

| クラス | 用途 |
|--------|------|
| `p-1` | メニュー項目、小要素 |
| `p-2` | リスト項目、タグ |
| `p-4` | カード、ポップオーバー |
| `p-6` | モーダル、大きなカード |
| `p-8` | ページコンテナ |

---

## コンポーネント別パターン

### ボタン

| サイズ | Padding |
|--------|---------|
| Small | `px-2 py-1` |
| Default（標準） | `px-4 py-2` |
| Large | `px-6 py-2` |

### カード

| サイズ | Padding | 内部 gap |
|--------|---------|----------|
| 標準 | `p-4` | `gap-1` |
| 大きなカード | `p-6` | `gap-2` |

### リスト項目

`px-4 py-2` または `px-2 py-2`

### モーダル / ダイアログ

- Padding: `p-6`
- セクション間: `mt-6`
- ボタン間: `gap-2`

### フォーム

- ラベル-入力間: `gap-1`
- フィールド間: `gap-4`

---

## 入れ子の Spacing

内側→外側に向かって Spacing を大きくする。

```
外側: p-8 (32px)
  └─ 中間: p-6 (24px) / セクション間: gap-6
       └─ 内側: p-4 (16px) / 項目間: gap-4
            └─ 最内側: p-2, gap-2 (8px)
```

**ルール**: 内側の Spacing は外側より小さく。

---

## 避けるべきパターン

- ✗ 一様な Spacing（関係性が伝わらない）
- ✗ 内側 > 外側の Spacing
- ✗ 任意値 `p-[13px]`
- ✗ スケール外 `0.5, 1.5, 3, 5, 7`
