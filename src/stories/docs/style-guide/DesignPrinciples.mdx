# デザイン原則

Typography・Motion・Layout の要約。詳細は各 Tokens/* を参照。

---

## Typography

フォント: Inter（英語）+ Noto Sans JP（日本語）。next/fontで最適化済み。フォント変更の提案は禁止。

| 用途 | クラス | サイズ |
|------|--------|--------|
| ページタイトル | `text-2xl font-bold` | 24px |
| セクション見出し | `text-lg font-bold` | 18px |
| カード見出し | `font-bold` | 16px |
| 本文 | `text-base` | 16px |
| UIラベル | `text-sm` | 14px |
| 補助テキスト | `text-xs text-muted-foreground` | 12px |

**ウェイトは2種類のみ**:

- ✅ `font-bold` — 見出し、強調
- ✅ `font-normal` — 本文（デフォルト）
- ❌ `font-black`, `font-extrabold`, `font-semibold` — 禁止
- ❌ `text-[17px]` — 任意値は禁止

> 詳細は [Tokens/Typography](?path=/story/tokens-typography--overview) を参照

## Motion

機能的アニメーションのみ許可。装飾的アニメーションは禁止。

| 種類 | 用途 | 実装 |
|------|------|------|
| Transition | ホバー、フォーカス | `transition-colors duration-150` |
| Fade | 表示/非表示 | `transition-opacity duration-200` |
| Scale | 押下フィードバック | `active:scale-[0.98]` |

**標準パターン**:

```tsx
// 標準トランジション（150ms）
transition-colors duration-150

// ホバー + 押下
transition-colors hover:bg-state-hover
active:scale-[0.98]

// motion-reduce 対応
motion-reduce:transition-none
```

**禁止されるアニメーション**:

- ❌ ページ遷移アニメーション（速度優先）
- ❌ 背景パーティクル（装飾的）
- ❌ バウンス、弾性（生産性ツールに不適）
- ❌ ローディングの凝った演出

> 詳細は [Tokens/Motion](?path=/story/tokens-motion--overview) を参照

## Layout

Google Calendar / Toggl と同等の情報密度を目指す。

| コンポーネント | 密度 | 例 |
|---------------|------|-----|
| カレンダーセル | 高密度 | 多くのイベント表示可能 |
| リスト項目 | 中密度 | 必要情報のみ |
| フォーム | 低密度 | 操作性優先 |

**コンポーネント選択フロー**:

```
新しいUIコンポーネントが必要か？
├─ shadcn/ui にあるか？ → あれば使う（第一選択）
├─ HeadlessUI にあるか？ → 使う（a11y重視）
├─ 3箇所以上で使うか？ → components/common/ に追加
└─ 1-2箇所のみ？ → インラインで実装
```
