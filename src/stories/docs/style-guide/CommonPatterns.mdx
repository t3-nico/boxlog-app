# 共通パターン

再利用可能なUIパターンとコード例

---

## WarningBox（アウトラインパターン）

破壊的操作の警告に使用。

```tsx
<div className="border-warning text-warning flex items-start gap-3 rounded-lg border p-4">
  <AlertTriangle className="size-5" />
  <div>この操作は取り消せません</div>
</div>
```

---

## InfoBox（Container背景）

使用状況、説明、詳細情報に使用。

```tsx
<div className="bg-container rounded-lg p-4">
  <p className="text-sm font-bold mb-2">使用状況:</p>
  <ul className="text-muted-foreground space-y-1 text-sm">
    <li>Plans: 10件</li>
  </ul>
</div>
```

---

## 余白設計（関連性ベース）

要素間の関連性が強いほど余白を小さく。

| サイズ | クラス | 用途 |
|--------|--------|------|
| 4px | `gap-1` | アイコン+テキスト、ラベル+入力 |
| 8px | `gap-2` | 同グループ内（ボタン群、タグ並び） |
| 16px | `gap-4` | フォームフィールド間、カード並び |
| 24px | `gap-6` | セクション間 |
| 32px | `gap-8` | 大きなセクション区切り、ナビとコンテンツ |

---

## AlertDialog パターン

確認ダイアログの標準レイアウト。

```tsx
<AlertDialog>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>タイトル</AlertDialogTitle>
    </AlertDialogHeader>
    {/* WarningBox + InfoBox */}
    <AlertDialogFooter>
      <AlertDialogCancel>キャンセル</AlertDialogCancel>
      <AlertDialogAction>実行</AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```
