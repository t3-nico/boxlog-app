# 📦 アーカイブドキュメント

このディレクトリには、過去の実装記録や古くなったドキュメントを保管しています。

## 📁 ディレクトリ構成

```
archive/
├── completed/           # 完了した実装の記録
├── development/         # 古い開発ドキュメント
├── analysis/            # 調査・分析レポート
├── reports/             # 品質監査レポート
├── compliance/          # 古いコンプライアンスルール
└── refactoring/         # 完了したリファクタリング記録
```

## ⚠️ 注意事項

- アーカイブ内のドキュメントは**参照専用**です
- 最新の情報は各カテゴリの現行ドキュメントを参照してください
- 古いコードパターンや設定は現在と異なる場合があります

## 📋 アーカイブ対象基準

以下の条件に該当するドキュメントをアーカイブ：

1. **完了したフェーズ**: Phase X など特定時点の実装記録
2. **古いデータ**: 3ヶ月以上更新のない統計・分析
3. **置き換え済み**: 新しいドキュメントで代替されたもの
4. **一時的な調査**: 1回限りの調査レポート
