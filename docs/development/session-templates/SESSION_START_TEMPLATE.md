# セッション開始テンプレート

Claude Codeセッション開始時に使用するテンプレートです。

## 🎯 基本テンプレート

```bash
echo "================================"
echo "🎯 セッション開始"
echo "================================"
echo "📅 日時: $(date)"
echo "🏷️  Issue: #XXX - [Issue タイトル]"
echo "🎯 目標: [このセッションの具体的目標]"
echo "⏱️  予定時間: [時間]"
echo "📋 工程: [調査/設計/実装/検証]"
echo "🔗 関連: [関連Issue・PR番号]"
echo "================================"
```

## 📊 使用方法

### 1. テンプレート選択

適切な工程・状況に応じてテンプレートを選択

### 2. 情報入力

`[XXX]`部分を実際の情報に置き換え

### 3. セッション開始記録

Claude Codeセッション開始時に実行

### 4. 目標設定確認

設定した目標が具体的で測定可能かを確認
