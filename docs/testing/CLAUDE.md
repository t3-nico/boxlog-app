## テスト作成クイックリファレンス

### コロケーション方式（必須）

```
src/features/tasks/
├── components/
│   ├── TaskList.tsx
│   └── TaskList.test.tsx  ← コンポーネントの隣に配置
├── stores/
│   ├── useTaskStore.ts
│   └── useTaskStore.test.ts
└── utils/
    ├── taskHelpers.ts
    └── taskHelpers.test.ts
```

### テスト対象の優先度

| 優先度 | 対象                                     |
| ------ | ---------------------------------------- |
| 高     | ビジネスロジック、認証・認可、データ保存 |
| 中     | カスタムフック、ユーティリティ関数       |
| 低     | 定数定義、型定義                         |

### フレームワーク

- **Vitest** — テストランナー
- **@testing-library/react** — コンポーネントテスト
- **@testing-library/user-event** — ユーザーインタラクション
- **Storybook play関数** — UIインタラクションテスト
