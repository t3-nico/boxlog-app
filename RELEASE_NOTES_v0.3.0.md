# BoxLog v0.3.0 Release Notes

## 🎉 概要

v0.3.0では、Tags機能の完全実装とGoogleドライブ風のUI/UX改善を中心に、大規模なアップデートを実施しました。36個のコミット、1000行以上のコード追加により、ユーザーエクスペリエンスが大幅に向上しています。

---

## ✨ 主な新機能

### 1. Tags機能の完全実装

#### Googleドライブ風の選択バー

- **複数選択対応**: チェックボックスまたは右クリックで複数のタグを選択可能
- **統一的な選択バー**: 選択時に自動的に表示されるアクションバー
- **共通アクション**: グループ移動、アーカイブ、削除を一括実行
- **選択解除**: ×ボタン、空白部分クリック、チェックボックス再クリックで解除

#### インライン編集

- タグ名をテーブル内で直接編集可能
- Enter キーで保存、Escape キーでキャンセル
- モーダルレスな編集体験

#### ドラッグ&ドロップ

- サイドバーのグループ間でタグをドラッグ&ドロップ可能
- ドラッグ中の視覚的フィードバック
- 直感的なグループ管理

#### コンテキストメニューとドロップダウンメニューの統合

- 右クリックメニューと選択バーのメニューを共通化
- コンテキストメニューを開いたまま選択バーを操作可能（Googleドライブと同じUX）
- z-indexの最適化により、両方のメニューが同時に表示可能

#### テーブル機能の強化

- **カラムリサイズ**: ドラッグで列幅を調整可能
- **横スクロール**: 多数の列でも快適に表示
- **グループ列**: タグ数を表示
- **スマートなプレースホルダー**: ホバー時のみ「説明を追加...」「グループを追加...」を表示

#### 国際化(i18n)完全対応

- Tags機能全体で日本語・英語に対応
- HTML lang属性を動的に設定（`/ja/`, `/en/`）
- サイドバー、テーブル、ダイアログすべてで翻訳対応

### 2. Inbox機能

- InboxTableView完全実装
- カスタムView作成機能
- 高度なテーブル機能統合

### 3. plan Inspector

- UI/UX大幅改善
- 統合的なインスペクター機能

### 4. 認証基盤

- **RLS完全実装**: Row Level Securityによるデータ保護
- **Supabase認証**: 完全な認証フロー実装
- **パフォーマンス最適化**: 認証処理の高速化

### 5. 状態管理の標準化

- Zustand + TanStack Query + Context API の統一的な使用
- 全アプリケーションで一貫した状態管理パターン

### 6. ライセンス管理

- 完全自動化
- コンプライアンス強化
- ライセンス情報の自動収集と表示

## 🏗️ アーキテクチャ改善

### コンポーネント構成

#### 新規コンポーネント

- `TagsSelectionBar.tsx` - Googleドライブ風の選択バー
- `TagSelectionActions.tsx` - 選択バーのアクションボタン群
- `TagActionMenuItems.tsx` - メニュー項目の共通コンポーネント（render props パターン）
- `TagsPageHeader.tsx` - 統一的なページヘッダー

#### グローバル変更

- `context-menu.tsx`:
  - `border-border` クラス追加でボーダー色を統一
  - z-index を `z-50` → `z-[9998]` に変更して DropdownMenu と共存

#### 削除されたコンポーネント

- `TagEditModal.tsx` - インライン編集に置き換え

### レイアウト統一

- **AppBarとSidebarの統一**: 一貫したレイアウト
- **ヘッダー高さ**: 48px固定（8px top padding + 40px container）
- **横幅パディング**: メイン 24px、ヘッダー 16px で統一
- **8pxグリッドシステム**: すべてのスペーシングで適用
